First, run the development server:

```bash

# or
```

## IP Allowlist (workers.dev)

Requests hitting the workers.dev endpoint are filtered via middleware. Provide allowed IPs or CIDR ranges with the `IP_ALLOWLIST` environment variable (comma separated). Example:

```bash
wrangler secret put IP_ALLOWLIST
# enter value: 203.0.113.5,198.51.100.0/24,2001:db8::/48
```

`IP_ALLOWLIST` が空、または有効なエントリが 1 つもない場合は全てのリクエストを許可します。値を更新する場合は再デプロイ、もしくはシークレットの再設定を行ってください。
